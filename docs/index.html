🔬 sci_paper_llm

Build datasets of open-access scientific papers from OpenAlex, and ask questions about them with a local LLM — all on your machine.

View on GitHub Open in Codespaces

What it does

Type a research topic and a date range; it pulls matching open-access papers from OpenAlex, downloads their full text, and builds a tidy dataset (authors, title, content, date, country, journal, theme). Then ask questions in plain language — answered by your local Ollama model, grounded in the downloaded papers with citations.

Features

🔎 Topic + date search

Query any field with an optional date bracket.

⬇️ Parallel downloads

Concurrent PDF fetch + text extraction with a live progress bar.

🏷️ Auto themes

A research-theme tag per paper, no API key required.

💬 Local Q&A

Answers from your own Ollama model, with source citations.

🐳 Dockerised

Bundled Ollama + app via Docker Compose.

☁️ Codespaces-ready

Runs free in a GitHub Codespace.

How it works

OpenAlex API → metadata (authors, title, date, journal, country, theme)
→ download PDFs (parallel, open-access only) → PyMuPDF text → content
→ pandas DataFrame → papers.parquet / papers.csv
→ ask a question → retrieve relevant papers → Ollama answer + citations

Run it

Fully self-contained with Docker Compose (bundles Ollama + the app):

git clone https://github.com/RebornX10/sci_paper_llm
cd sci_paper_llm
docker compose up        # then open http://localhost:8000

This is a self-hosted application — run it locally, with Docker, or in a Codespace. It needs a live Python server and Ollama, so it can't run on GitHub Pages (this page is just the project's front door).
